
### Redox and Electrochem Panic Block

```text
OIL RIG
Oxidation Is Loss
Reduction Is Gain
```

```text
LEO GER
Lose Electrons = Oxidation
Gain Electrons = Reduction
```

```text
Anode = Oxidation
Cathode = Reduction
Electrons flow Anode → Cathode
```

```text
Reducing agent gets oxidized and helps reduce another species.
Oxidizing agent gets reduced and helps oxidize another species.
```

```text
RAO: Reducing Agent Oxidized
OAR: Oxidizing Agent Reduced
```

```text
Ecell = Ered,cathode − Ered,anode
```

```text
Ecell > 0 → ΔG < 0 → spontaneous
```

```text
ΔG = −nFE
```

```text
+ E°red = better reduced
− E°red = better oxidized
```

Clarification:

```text
Ecell = whole-cell potential
E°red = half-cell reduction potential
```

### Thermodynamics and Sign Block

```text
ΔH > 0 = heat absorbed = endothermic
ΔH < 0 = heat released = exothermic
```

```text
ΔG < 0 = spontaneous / free-energy downhill
ΔG > 0 = nonspontaneous / free-energy uphill
```

```text
ΔG = ΔH − TΔS
```

```text
ΔG = ΔG° + RT ln Q
```

```text
Q < 1 → ln Q < 0 → ΔG decreases
Q > 1 → ln Q > 0 → ΔG increases
```

```text
R = energy-per-mole-per-K constant, not gas-only
```

### Bond Enthalpy Block

```text
Break bonds = pay energy = +
Form bonds = get energy back = −
```

```text
ΔHrxn = bonds broken − bonds formed
```

Better mental model:

```text
ΔHrxn = (+ break) + (− form)
```

### Acid/Base and Log Block

```text
pH = −log[H+]
↑[H+] → ↓pH
↓[H+] → ↑pH
```

```text
High pH = low H+ = more basic
Low pH = high H+ = more acidic
```

```text
pKa = −log Ka
↑Ka → ↓pKa → stronger acid
```

```text
Low pKa = strong acid
High pKa = weak acid
```

```text
pH = pKa + log(A−/HA)
pH = pKa → A− = HA
```

### Periodic Trends Quick Block

```text
IE increases up/right
EA generally more negative up/right, with exceptions
```

```text
Successive IE increases
Big IE jump = core electron removal started
```

```text
Be > B for IE
N > O for IE
```

```text
Isoelectronic ions: more protons = smaller ion
```

### Units and Constants Block

```text
n = 10^-9
μ = 10^-6
m = 10^-3
k = 10^3
```

```text
h ≈ 6.6 × 10^-34 J·s
c ≈ 3 × 10^8 m/s
e ≈ 1.6 × 10^-19 C
g ≈ 10 m/s²
```

Personal unit warning:

```text
kg ≠ g
mg ≠ μg
Track units before plugging in.
```

### CARS Rules

```text
No outside knowledge.
```

```text
Main idea: 7–12 words.
```

```text
Tone: + / − / neutral / mixed.
```

```text
Stem verb = task.
```

```text
Use exact line + before/after.
```

```text
Extreme answer needs proof.
```

```text
If stuck: general answer > weird specific answer.
```

Personal trust reminder:

```text
I do not need every detail to answer correctly.
```

Stuck strategy:

```text
Reread exact sentence.
Check previous/next sentence.
Use verbs in the stem.
Pick the answer the passage can prove.
```

```text
60=Q37, 70=Q43, 80=Q50, 90=Q56, 95=Q59

```

### Experiment Attack Block

```text
What changed?
```

```text
Control vs experimental?
```

```text
Independent variable → dependent variable
```

```text
↑ / ↓ / no change
```

```text
Gene → mRNA → protein → function
```

```text
Knockout / inhibit = ↓ function
Overexpress / activate = ↑ function
```

### Enzyme Block

```text
Km = [S] at 1/2 Vmax
```

```text
↑Km = ↓affinity
```

```text
Competitive: ↑Km, Vmax same
Noncompetitive: Km same, ↓Vmax
```

### Amino Acid and Charge Block

```text
Acidic AAs: D, E = −
Basic AAs: K, R, H = +
```

```text
DE = negative acid side chains
KRH = positive basic side chains
```

```text
Amino acid charge mnemonic:
Dragons Eat negative acid.
Knights Ride Horses positive basic.
```

```text
Nonpolar core: GAVLIMP
Aromatic: FYW
Hydroxyl: STY
Sulfur: C, M
Amide: N, Q
```

```text
pH > pI → protein more negative
pH < pI → protein more positive
```

```text
High pH pulls off H+ → molecule/protein becomes more negative.
Low pH adds H+ → molecule/protein becomes more positive.
```

### Molecule-Family Block

```text
Scaffold + modification = function
```

```text
ATP = adenine + ribose + 3 phosphates
```

```text
ATP vs dATP: ribose vs deoxyribose
```

```text
NADH / NADPH / FADH2 = electron carriers
```

```text
Acetyl-CoA = acetyl delivery
```

```text
Ask: what changed in structure, and what changed in function?
```

### P/S Trap Terms

```text
Availability = easy to recall
Representativeness = matches prototype
Confirmation bias = seek confirming evidence
```

```text
Fundamental attribution error = overuse disposition for others
Self-serving bias = credit self, blame situation
```

```text
Halo effect = one trait affects whole judgment
Physical attractiveness stereotype = attractive judged as better overall
```

```text
Role strain = tension within same role
Role conflict = tension between different roles
```

```text
Groupthink = harmony over good decisions
Conformity = behavior changes to match group
Compliance = behavior changes after request
Obedience = behavior changes after authority command
```
