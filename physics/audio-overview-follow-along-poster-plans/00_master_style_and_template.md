# Physics Audio Overview Follow-Along Poster System — Master Style and Template

## Build principle

These are not textbook summaries. They are audio-follow-along visual blueprints. Each poster should help the listener keep pace with the podcast while seeing the structure, formula logic, graph relationships, and MCAT traps.

## Visual style

- White background is acceptable and preferred.
- Navy header bar or navy title treatment.
- Blue, teal, yellow, red, green, and purple accent blocks.
- Large central visual first, text second.
- Formula boxes should be big enough to read on iPad or laptop.
- Use an MCAT Trap strip on every poster.
- Use an Audio-Follow Cue strip on every poster.
- Avoid tiny dense text.
- Use diagrams, arrows, graphs, and comparison tables.

## Universal poster template

```md
# Poster [Number] — [Poster Title]

## Audio purpose
[What this poster helps the listener track in the audio.]

## Main visual
[Large central diagram or graph.]

## Required formulas
- Formula 1
- Formula 2

## Visual elements
- Element 1
- Element 2
- Element 3

## MCAT trap strip
Trap: [common wrong interpretation]
Fix: [test-day correction]

## Audio-follow cue
Use this poster when the audio says: [keywords/phrases]

## Suggested filename
`Physics_Ch##_Audio_##_[Short_Name].png`
```

## Standard chapter file structure

Each chapter plan should include:

1. Recommended number of posters
2. Chapter-level audio journey
3. Poster-by-poster blueprint
4. Production checklist
5. Suggested PNG filenames
