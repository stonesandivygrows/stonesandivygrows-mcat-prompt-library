## 1. Alcohol chain length and water solubility

### Trigger

Anki card: short-chain alcohols are more soluble in water than long-chain alcohols.

### Correct rule

```text
Same number of -OH groups: longer carbon chain -> more hydrophobic character -> lower water solubility.
```

### Wrong path to avoid

Do not predict ordinary neutral-alcohol solubility by asking whether a longer alkyl chain stabilizes a conjugate acid.

That is acid/base reasoning, not the main solubility reasoning.

### Sticky version

```text
Alcohol solubility = polar -OH vs nonpolar carbon chain.
Longer chain = more hydrophobic = less water soluble.
```

## 2. Phosgene vs phosphorus

### Trigger

Confusion over why phosgene has chlorine atoms and no phosphorus.

### Correct rule

```text
phosgene = COCl2 = carbonyl dichloride
phosgene does not contain phosphorus
```

### Why the name is confusing

The "phos-" in phosgene refers to light-generated, not phosphorus-containing.

### Sticky version

```text
phosgene != phosphate
phosgene != phosphorus chemistry
phosgene = O=CCl2, a carbonyl dichloride
```

## 3. Thermodynamic stability vs kinetic stability

### Trigger

Peptide bonds felt contradictory because they are described as thermodynamically unstable but kinetically stable.

### Correct rule

```text
Thermodynamics = does it want to happen?
Kinetics = how fast does it happen?
```

### Peptide bond logic

```text
Peptide bond hydrolysis is thermodynamically favorable.
Therefore peptide bonds are thermodynamically unstable relative to free amino acids.

But peptide bond hydrolysis has high activation energy.
Therefore peptide bonds are kinetically stable.
```

### Rust analogy

```text
Rust is the lower-energy endpoint.
Rusting is thermodynamically favorable.
But rusting happens slowly, so it is kinetically slow/stable.
```

### Sticky version

```text
Thermo = want.
Kinetics = wait.

Thermodynamically unstable = wants to change.
Kinetically stable = waits a long time to change.
```

### Better Anki wording

```text
Peptide bonds are kinetically stable because peptide bond hydrolysis has a high activation energy.
```

## 4. Basement membrane vs connective tissue ECM

### Trigger

Confusion over whether the extracellular matrix of connective tissue is called the basement membrane.

### Correct rule

```text
Basement membrane = specialized ECM layer at the base of epithelial tissue.
Connective tissue ECM = broad support material in connective tissue.
```

### Tissue map

```text
epithelial cells
↓
basement membrane
↓
connective tissue with abundant ECM
```

### Sticky version

```text
Basement membrane is not all connective tissue ECM.
It is the thin specialized ECM layer that anchors epithelium to underlying connective tissue.
```

### Better Anki wording

```text
The basement membrane is a specialized extracellular matrix layer that anchors epithelial tissue to underlying connective tissue.
```

## 5. Histidine and pI

### Trigger

Question: why does histidine increase pI less than lysine or arginine?

### Correct rule

```text
Higher side-chain pKa = holds H+ longer = stays positive longer = raises pI more.
Lower side-chain pKa = gives up H+ earlier = becomes neutral earlier = raises pI less.
```

### Basic amino acid order

```text
Arginine > Lysine > Histidine
highest pKa -> lowest pKa
most pI increase -> least pI increase
```

### Histidine detail

Histidine deprotonation is:

```text
His-H+ ⇌ His + H+
positive    neutral
```

Do not think histidine deprotonates into a negatively charged conjugate base. The deprotonated imidazole is neutral.

### Sticky version

```text
Histidine is basic, but conditionally basic.
It loses positive charge around pH ~6, so it raises pI less than Lys or Arg.
```

## 6. Amino acids that commonly act as nucleophiles

### Trigger

Question: why are Ser, Thr, Tyr, Cys, Lys listed, but not Arg or His?

### Common MCAT list

```text
Ser, Thr, Tyr, Cys, Lys
S, T, Y, C, K
```

### Why these count

```text
Ser/Thr/Tyr: oxygen nucleophiles when deprotonated
Cys: sulfur nucleophile when deprotonated
Lys: nitrogen nucleophile when deprotonated
```

### Why arginine usually does not count

```text
Arginine = guanidinium group
mostly protonated and resonance-stabilized
poor attacking nucleophile
```

### Why histidine is special

```text
Histidine can sometimes be nucleophilic,
but on the MCAT it is more commonly tested as an acid/base catalyst.
```

### Sticky version

```text
Common side-chain nucleophiles = S, T, Y, C, K.
Arg = charged stabilizer/salt bridges.
His = proton shuttle/general acid-base helper.
```

## 7. Amphoteric vs amphiprotic vs amphipathic

### Trigger

Confusion between similar "amphi-" terms.

### Correct map

```text
Amphoteric = can act as acid or base.
Amphiprotic = can donate or accept H+.
Amphipathic = has hydrophilic and hydrophobic regions.
```

### Relationship

```text
All amphiprotic compounds are amphoteric.
Not all amphoteric compounds are amphiprotic.
```

### Sticky version

```text
-theoteric = acid/base behavior
-protic = proton behavior
-pathic = polarity/solubility behavior
```

### Better Anki wording

```text
Amphoteric compounds can act as either acids or bases.
Amphiprotic compounds are amphoteric compounds that specifically can either donate or accept H+.
Amphipathic compounds have both hydrophilic and hydrophobic regions.
```

```text
Syncretism = beliefs/religions/practices blend.
Amalgamation = groups/cultures/peoples blend into a new combined whole.
```

## 9. Ring/group hooks: pyrrolidine, guanidinium, imidazole, indole, pyrrole

### Main map

```text
Proline -> pyrrolidine
Arginine -> guanidinium
Histidine -> imidazole
Tryptophan -> indole
Heme/porphyrin -> pyrrole
```

### Structure hooks

```text
Pyrrolidine = saturated 5-member ring with 1 nitrogen = Proline
Guanidinium = carbon with 3 nitrogens = Arginine, very basic/positive
Imidazole = 5-member aromatic ring with 2 nitrogens = Histidine
Indole = fused 5-member N ring + benzene = Tryptophan
Pyrrole = aromatic 5-member ring with 1 nitrogen = heme/porphyrin unit
```

### Sticky chain

```text
Proline is locked -> pyrrolidine.
Arginine is charged -> guanidinium.
Histidine switches protons -> imidazole.
Tryptophan is huge -> indole.
Heme holds iron -> pyrrole rings.
```

## 10. Reducing end vs nonreducing end of sugars

### Trigger

Question: is reducing end right and nonreducing end left?

### Correct rule

```text
Reducing end = free anomeric carbon.
Nonreducing end = anomeric carbon locked in glycosidic bond.
```

### Drawing shortcut

In many textbook drawings:

```text
left = nonreducing end
right = reducing end
```

But do not trust left/right. Trust the anomeric carbon.

### Sucrose exception

```text
Sucrose is nonreducing because both anomeric carbons are locked in the glycosidic bond.
```

### Sticky version

```text
Free anomeric carbon = reducing end.
Locked anomeric carbon = nonreducing end.
```

## 12. D-glucose vs L-glucose: enantiomers, not epimers

### Trigger

Anki card claimed L-glucose and D-glucose are epimers and enantiomers.

### Correction

```text
D-glucose and L-glucose = enantiomers.
D-glucose and L-glucose ≠ epimers.
```

### Why

```text
Enantiomers = differ at all chiral centers.
Epimers = differ at exactly one chiral center.
Diastereomers = differ at some but not all chiral centers.
```

Since D-glucose and L-glucose differ at all chiral centers, they are enantiomers.

Since epimers are diastereomers that differ at exactly one chiral center, D-glucose and L-glucose cannot be epimers.

### Classic epimer examples

```text
D-glucose vs D-mannose = C-2 epimers.
D-glucose vs D-galactose = C-4 epimers.
```

### Sticky version

```text
All centers flipped = enantiomers.
One center flipped = epimers.
Some but not all centers flipped = diastereomers.
```

### Better Anki wording

```text
L-glucose and D-glucose are enantiomers because they differ in configuration at all chiral centers.
```

Extra:

```text
Epimers are diastereomers that differ at exactly one chiral center, such as D-glucose/D-mannose or D-glucose/D-galactose.
```
