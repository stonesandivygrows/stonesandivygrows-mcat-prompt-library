# MCAT Prompt Library

Private repository for organizing MCAT study prompts by tool and use case.

## Folder structure

- `notebooklm/audio/` — chapter and lecture prompts for NotebookLM audio output
- `notebooklm/visuals/` — visual-output and follow-along prompt templates
- `notebooklm/behavioral-science/` — Khan Academy and Psych/Soc group prompts
- `macwhisper/` — transcription, note conversion, and title-generation prompts
- `canva/` — transcript-to-visual and design-generation prompts
- `cars/` — CARS-specific prompts and review frameworks
- `naming-rules/` — transcript title and naming conventions
- `archive/` — legacy or superseded prompt versions
