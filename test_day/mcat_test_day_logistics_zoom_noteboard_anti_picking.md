# MCAT Test-Day Logistics: Zoom, Noteboard, and Anti-Picking Strategy

## Status

Working test-day logistics note based on MCAT tutorial practice and Pearson VUE erasable noteboard reference.

## Core discovery: zoom out reduces overwhelm

During the MCAT tutorial, I confirmed that zooming in and out is available in the testing interface. Zooming out is helpful because it makes passages feel less intimidating.

When the screen is zoomed in, the text feels long and overwhelming. When zoomed out, many science and CARS passages look more manageable, often only a few paragraphs.

### Test-day rule

Use a zoom level that keeps more of the passage visible.

Prefer:
- zoomed-out passage view
- less scrolling
- leaning forward if needed
- selective highlighting only

Avoid:
- excessive scrolling up and down
- highlighting mainly because paragraph locations are hard to see
- letting the passage look larger than it actually is

## Why this matters for CARS and science passages

Highlighting has been useful as a mental map because the shape of highlighted text can help identify a paragraph later. Example: a long highlighted sentence can create a visual shape that makes the paragraph easier to relocate.

However, if zooming out lets more paragraphs stay visible, I do not need to rely on highlighting as heavily.

### New reading setup

- Use zoomed-out view as the first mental map.
- Use highlighting only for high-value anchors.
- Do not highlight just to compensate for scrolling confusion.
- Use paragraph layout and passage shape to stay oriented.

## Pearson VUE noteboard reference

The uploaded Pearson VUE material identifies the testing scratch surface as an erasable booklet/noteboard and includes an erasable grid-style noteboard page.

Important instructions from the Pearson VUE page:
- Raise hand and wait for the test administrator for breaks, hardware/software problems, illness, disturbances, or other assistance.
- Personal items are not allowed in the testing room.
- Eating, drinking, smoking, talking, or disturbing other candidates is prohibited.
- The erasable booklet and other exam materials may not be used before the exam starts.
- Removing an erasable booklet from the testing area is misconduct.
- Bring ID when exiting and returning from breaks.

## Noteboard strategy

Because the noteboard is erasable, visual clutter can be managed better than on plain paper.

### Default rule

Cross out small mistakes.

### Use erasing only when needed

Erase only when visual clutter would interfere with reading or timing.

Examples:
- timing box got messy
- calculation area became unreadable
- crossed-out notes are blending together
- wrong setup is visually distracting

### Thumb/sleeve backup

If the noteboard behaves like wet-erase, a thumb or sleeve edge may work for small corrections.

However:
- Do not depend on this as the main system.
- Do not create a habit loop around erasing.
- Do not erase large sections during timed work unless necessary.
- If the marker behaves like permanent marker or does not erase cleanly, stop trying and cross out instead.

## Anti-picking and anti-fidgeting test-day prep

These are not vanity tasks. They are attention-protection tasks.

Physical triggers removed before the test means fewer urges during the test. Fewer urges means less attentional leakage and more working memory available for questions.

### Before entering the test center

- Braid hair so there is less urge to touch or pick at it.
- Trim nails.
- Remove or smooth every hangnail.
- Exfoliate lips so there is less to pick at.
- Pluck chin hairs beforehand.
- Apply lotion to hands after washing if dry hands create nervousness or a picking/fidgeting urge.
- Consider bringing tweezers only for permitted locker/break access, not for the testing room.
- Consider long sleeves for comfort and as a possible noteboard cleanup backup if allowed.

### During exam

- Keep hands task-directed: mouse, keyboard, noteboard, or lap.
- If hands feel dry after washing during breaks, use permitted lotion before returning if allowed by test-center rules.
- If picking urge appears, redirect to the noteboard/timing anchor.
- Do not spend mental energy negotiating with the urge.
- Use the rule: notice → redirect → continue.

## Chair and body-position logistics

I test better when my body feels safe and contained.

### Check before test day if possible

- What type of chair does the testing site use?
- Are there armrests?
- Can I comfortably cross my legs?
- Can my legs fit under the armrests if needed?

### Test-day body-position rule

Use the most secure comfortable posture available without wasting time adjusting repeatedly.

## Medication and break logistics

Bring necessary prescribed medication, especially blood pressure medication, according to official MCAT/test-center rules.

Medication should be planned as part of break logistics, not improvised during the exam.

### Rule

Confirm what can be accessed during breaks and what must stay stored.

Do not assume medication or personal items can enter the testing room unless officially allowed.

## Final compact checklist

### Night before / morning of

- Hair braided
- Nails trimmed
- Hangnails removed
- Lips exfoliated
- Chin hairs plucked
- Hands moisturized / lotion packed if allowed for locker or break access
- Medication packed according to official rules
- ID ready
- Long sleeve considered
- Tweezers only if allowed for locker/break use

### At tutorial / start

- Set comfortable zoom level
- Confirm passage visibility
- Do not over-highlight
- Remember: zoomed-out view is the first map

### During section

- Cross out small mistakes
- Erase only if clutter is hurting performance
- Use noteboard timing anchors
- Redirect picking urges immediately
- Lean forward rather than scroll repeatedly

## Core rule

Reduce physical friction before the test so the exam is only about reasoning, not managing preventable distractions.
