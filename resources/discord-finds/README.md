# Discord Resource Collection

**Purpose:** Collect high-value study materials from MCAT Discord communities.  
**Scope:** PDFs, Google Docs, spreadsheets, articles, practice tools, graphics. No social media links, no video links, no personal conversation.

## Servers Tracked

| File | Server | Focus |
|------|--------|-------|
| [aamc-warriors.md](aamc-warriors.md) | AAMC Warriors | Official AAMC resources, FL analysis, score discussion |
| [mi-gente-latino.md](mi-gente-latino.md) | Mi Gente Latino | Community resources, bilingual support, study guides |
| [528.md](528.md) | 528 | High-score strategy, subject-specific resources |

## What Belongs Here

- ✅ Google Drive/Docs links to study guides, spreadsheets, subject reviews
- ✅ Direct PDF links or downloads (save to Google Drive: `MCAT > Discord Finds > [server]`)
- ✅ Notion, Anki shared decks, GitHub repos shared in channels
- ✅ Practice tool links (question banks, timing tools, scheduling templates)
- ✅ Graphics and infographics (save to Drive, link from here)
- ❌ Discord message links (ephemeral, require login)
- ❌ YouTube/video links
- ❌ Personal member handles or conversation excerpts

## Google Drive Organization

Recommended Drive path for graphics/images that don't belong in GitHub:  
`MCAT > Discord Finds > [server-name] > [topic]`

## Chrome Tab Group Suggestion

Once high-use links are identified, create a Chrome Tab Group called **MCAT Discord** with pinned tabs for the most-used resources per server. Export that group's URLs here for permanence.
