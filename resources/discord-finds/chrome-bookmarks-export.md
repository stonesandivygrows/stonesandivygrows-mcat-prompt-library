## Chrome Tab Group Suggestion

Once the most-used links are identified, create a persistent **Tab Group** in Chrome:

1. Open the key resource tabs
2. Right-click any tab → **Add to group** → name it `MCAT Discord`
3. Pin the group so it survives browser restarts
4. Consider saving the group (Chrome → Bookmarks → Bookmark all tabs) as a folder

**Candidates for pinned tabs** (populate as links are found):
- AAMC Warriors #resources channel
- Mi Gente Latino pinned resources
- 528 #resources channel
- Any shared Google Drive folders

## How to Export Current Bookmarks Here

When you have links collected:

1. Copy each URL from Chrome
2. Add to the appropriate server file ([aamc-warriors.md](aamc-warriors.md), [mi-gente-latino.md](mi-gente-latino.md), [528.md](528.md))

This file is a template and guide — the actual links live in the per-server files.
