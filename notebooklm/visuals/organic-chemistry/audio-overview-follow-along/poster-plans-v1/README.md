# Organic Chemistry Audio Overview Follow-Along Poster Plans

Status: planning layer only. Do not generate or replace images from this folder until a chapter plan is approved.

Purpose: store chapter-by-chapter poster plans for MCAT Organic Chemistry audio overviews before any image generation. The audio transcript remains the primary ordering source. Kaplan chapter structure is the spine. UWorld, Anki, and past mistakes are support layers.

## Output standard per chapter

Each chapter plan should include:

- One audio navigation / overview poster
- Subsection follow-along posters in lecture order
- Mechanism rescue posters when curved-arrow logic is dense
- Structure recognition posters when molecules or functional groups are central
- MCAT trap cards for common mix-ups
- Suggested file names for eventual PNG generation
- Notes about which existing poster/image set, if any, already exists

## Visual style standard

Use the visual style Neissa has preferred from prior MCAT poster sets:

- White or clean light background
- Navy chapter headers
- Purple organic chemistry accent palette, with teal/blue/yellow/red as functional cues
- Large central structures and reaction maps
- Substrate → reagent/condition → product arrows
- Curved-arrow mechanism panels when relevant
- Functional group callouts
- MCAT Trap strip
- Audio-Follow Cue strip
- Minimal text walls; concepts should be mapped visually

## Workflow rule

1. Store the poster plan on GitHub.
2. Review and approve the chapter plan.
3. Generate images one chapter at a time.
4. Save generated PNGs with the planned filenames.
5. Link final assets back to Notion / local visual manifest only after image review.

## No-delete policy

Do not overwrite or delete older plans. If a chapter plan needs revisions, create a dated or versioned file, for example:

- `Ch06_Aldehydes_Ketones_I_Poster_Plan_v2.md`
- `Ch06_Aldehydes_Ketones_I_Poster_Plan_2026-05-20.md`
