# AAMC + Jack Westin Extension Software Practice Note

## Context

This note captures the troubleshooting and strategy decision from the AAMC CARS Question Pack + Jack Westin Chrome extension discussion.

The issue was not mainly Chrome site permissions. The screenshots showed the Jack Westin extension was active on the AAMC Prep Hub, including the JW sidebar and an "Extension is Active" status. The mismatch came from expecting the extension to fully convert the AAMC Question Pack interface into the test-day interface.

## Main conclusion

Use each platform for its best purpose:

- **AAMC Question Packs:** best for official AAMC CARS logic, passage reasoning, answer-choice traps, and review.
- **AAMC full-length exams / scored practice exams:** best for test-day software simulation, timing, navigation, and interface familiarity.
- **Jack Westin extension:** best as a post-practice explanation/review aid, not as the primary test-day simulator.

## Practice protocol for AAMC CARS Question Packs

When using AAMC CARS Question Packs:

1. Click **Practice** from the AAMC item dashboard.
2. Treat the interface as official AAMC practice, even if it does not look like the full exam-day software.
3. Turn **Review Answer** off when doing a timed/authentic attempt.
4. Use manual timing by passage:
   - 5 questions: about 8 minutes.
   - 6 questions: about 9 minutes.
   - 7 questions: about 10 minutes.
   - 8 questions: about 11 minutes.
5. Use AAMC-native tools only during the attempt:
   - highlight selectively,
   - strikethrough/eliminate answer choices,
   - flag only questions worth returning to,
   - avoid relying on JW explanations until review.
6. After the timed attempt, then use the JW sidebar/explanations for review.

## Software-practice hierarchy

- **Highest fidelity:** AAMC full-length exams.
- **Medium fidelity:** AAMC section bank/question packs with manual timing.
- **Lower fidelity but useful:** Jack Westin/UWorld-style interfaces for daily stamina and keyboard/tool familiarity.

## Do / Don't

### Do

- Do use AAMC materials because they are written by the test maker.
- Do use full-length exams to practice the closest available software environment.
- Do keep the JW extension enabled if it helps review explanations.
- Do separate **question reasoning practice** from **software simulation practice**.

### Don't

- Don't assume the JW extension is broken just because the AAMC page does not become a full test-day clone.
- Don't spend too much study time troubleshooting extension behavior when the extension is already active.
- Don't rely on Question Packs alone for software/interface rehearsal.
- Don't let interface mismatch become a reason to avoid official AAMC practice.

## Operational rule

For CARS prep, the priority is:

**AAMC logic first, interface second.**

Use Question Packs to learn how AAMC asks and rewards reasoning. Use full-lengths to rehearse the actual test-day software rhythm.
