# Jack Westin CARS — Practice Logs

This folder stores **verbatim exports/transcriptions** of Jack Westin CARS practice sets.

## Rules
- **Website test date/title has priority** over filenames or personal labels.
- Preserve original wording/formatting as available.
- Use the **Full Template** format when possible (passage + questions + answer choices + selected/correct + full explanation). Keep **My Reasoning** blank.
- Dedupe: if a test/passage is already logged, do not re-add it.

## Structure
- `2026/` — one file per website test date (preferred)
- `archive/` — legacy combined dumps or partial OCR captures

## Naming convention (preferred)
`YYYY-MM-DD__JW_CARS__<Website_Test_Title_or_Short_Label>.md`

Examples:
- `2026-05-11__JW_CARS__Full_Export.md`
- `2026-04-13__JW_CARS__Full_Review.md`
