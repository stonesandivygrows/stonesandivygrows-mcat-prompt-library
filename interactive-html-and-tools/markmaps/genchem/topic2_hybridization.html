Topic 2: Hybridization & Quantum Numbers

The Two Key Quantum Numbers

Azimuthal Quantum Number (ℓ)

  • Determines the subshell type
  • Values: 0, 1, 2, 3 → corresponds to s, p, d, f
  • Range: 0 to (n-1)

Magnetic Quantum Number (mℓ)

  • Determines which specific orbital within that subshell
  • Values: -ℓ to +ℓ (includes 0)
  • Number of orbitals = (2ℓ + 1)
Subshell ℓ value mℓ values # of orbitals Max electrons
s 0 0 1 2

f 3 -3, -2, -1, 0, +1, +2, +3 7 14

Hybridization Explained

What is Hybridization?

Hybridization = mixing atomic orbitals to form new hybrid orbitals that are better for bonding

The Rule: Count electron domains

  • Electron domain = any region of electron density around an atom
  • 1 single bond = 1 domain
  • 1 double bond = 1 domain (even though it's 2 electrons)
  • 1 triple bond = 1 domain
  • 1 lone pair = 1 domain

The Three Main Hybridizations

Hybridization # Electron Domains Geometry Bond Angle Unhybridized p orbitals
sp 2 Linear 180° 2 remaining
sp² 3 Trigonal planar 120° 1 remaining
sp³ 4 Tetrahedral 109.5° 0 remaining

Example 1: sp³ Hybridization (Methane, CH₄)

  • Carbon has 4 bonds to H atoms
  • 4 electron domains → sp³
  • 1s + 3p orbitals mix → 4 sp³ hybrid orbitals
  • Each sp³ orbital holds max 2 electrons
  • All p orbitals used, so NO π bonds possible
  • 25% s character, 75% p character per hybrid orbital

Example 2: sp² Hybridization (Ethene, C₂H₄)

  • Each carbon has 3 σ bonds (2 to H, 1 to C)
  • 3 electron domains → sp²
  • 1s + 2p orbitals mix → 3 sp² hybrid orbitals (in a plane)
  • 1 p orbital remains unhybridized (perpendicular to plane)
  • This unhybridized p overlaps with the other C's p → forms π bond
  • Result: C=C double bond (1 σ + 1 π)
  • 33% s character, 67% p character per hybrid orbital

Example 3: sp Hybridization (Ethyne, C₂H₂)

  • Each carbon has 2 σ bonds (1 to H, 1 to C)
  • 2 electron domains → sp
  • 1s + 1p orbital mix → 2 sp hybrid orbitals (linear)
  • 2 p orbitals remain unhybridized
  • These 2 unhybridized p orbitals form 2 π bonds
  • Result: C≡C triple bond (1 σ + 2 π)
  • 50% s character, 50% p character per hybrid orbital

Electron Capacity Summary

🔑 Key Rules:

  1. Each orbital holds MAX 2 electrons (regardless of whether it's hybrid or not)
  2. Each subshell:
    • s = 1 orbital → max 2e⁻
    • p = 3 orbitals → max 6e⁻
    • d = 5 orbitals → max 10e⁻
    • f = 7 orbitals → max 14e⁻
  3. Hybridization doesn't change electron capacity - you're just reshaping the orbitals
  4. Unhybridized p orbitals are needed for π bonds in double/triple bonds

How to Determine Hybridization

Step-by-Step Method:

  1. Draw the Lewis structure
  2. Count electron domains around the atom:
    • Count each bond (single, double, OR triple) as 1 domain
    • Count each lone pair as 1 domain
  3. Match to hybridization:
    • 2 domains = sp
    • 3 domains = sp²
    • 4 domains = sp³

Examples:

  • NH₃ (ammonia): N has 3 bonds + 1 lone pair = 4 domains = sp³
  • H₂O (water): O has 2 bonds + 2 lone pairs = 4 domains = sp³
  • CO₂: C has 2 double bonds = 2 domains = sp
  • BF₃: B has 3 bonds = 3 domains = sp²
