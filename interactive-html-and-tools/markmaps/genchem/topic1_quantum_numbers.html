Topic 1: Shell vs Orbital vs Energy Level vs s/p/d/f

The Hierarchy (Biggest to Smallest)

  1. SHELL (n) = Energy level, distance from nucleus (n = 1, 2, 3, 4...)
  2. SUBSHELL (ℓ) = Shape type (ℓ = 0, 1, 2, 3 → named s, p, d, f)
  3. ORBITAL (mℓ) = Specific 3D orientation within a subshell
  4. ELECTRON = Individual electron with spin ↑ or ↓ (ms = +½ or -½)

Understanding Each Level

1. Shell (Principal Quantum Number, n)

  • Values: n = 1, 2, 3, 4, 5...
  • Represents the main energy level and distance from nucleus
  • Max electrons per shell = 2n²
    • n=1: max 2 electrons
    • n=2: max 8 electrons
    • n=3: max 18 electrons

2. Subshell (Angular Momentum Quantum Number, ℓ)

  • Values: ℓ = 0, 1, 2, 3... up to (n-1)
  • Determines the SHAPE of the electron cloud
  • s/p/d/f are just NAMES for these shapes:
ℓ value Subshell name Number of orbitals Max electrons
0 s (spherical) 1 2
1 p (dumbbell) 3 6
2 d 5 10
3 f 7 14

3. Orbital (Magnetic Quantum Number, mℓ)

  • Values: -ℓ to +ℓ (all integers in between)
  • Determines the ORIENTATION of the orbital in 3D space
  • Number of orbitals = (2ℓ + 1)
    • s subshell (ℓ=0): mℓ = 0 → 1 orbital
    • p subshell (ℓ=1): mℓ = -1, 0, +1 → 3 orbitals (px, py, pz)
    • d subshell (ℓ=2): mℓ = -2, -1, 0, +1, +2 → 5 orbitals
  • Each orbital holds MAX 2 ELECTRONS

4. Spin (Spin Quantum Number, ms)

  • Values: +½ (spin up ↑) or -½ (spin down ↓)
  • Two electrons in same orbital MUST have opposite spins
  • Pauli Exclusion Principle: No two electrons can have identical quantum numbers (n, ℓ, mℓ, ms)

Example: Carbon (6 electrons)

Electron configuration: 1s² 2s² 2p²

Let's look at the LAST electron (the 6th one):

  • n = 2 (it's in the 2nd shell)
  • ℓ = 1 (it's in a p subshell)
  • mℓ = 0 (it's in the middle p orbital, could be -1, 0, or +1)
  • ms = +½ (spin up)

Breakdown by shell:

  • Shell 1 (n=1): 2 electrons in 1s orbital
  • Shell 2 (n=2):
    • 2s subshell: 2 electrons
    • 2p subshell: 2 electrons (in 2 different orbitals, following Hund's rule)

🔑 Key Insight: What's the difference?

  • Shell = which floor of a building you're on
  • Subshell = the shape/type of room on that floor (square room vs round room vs L-shaped)
  • Orbital = a specific room of that shape (north-facing square room vs south-facing)
  • s/p/d/f = NOT a separate category! Just nicknames for subshell shapes

Common mistake: Thinking s/p/d/f is different from subshells. They're the SAME thing - just the names we use for different ℓ values!
