Quick visual memory: Glycerol has 3 OH groups (no double bond). Glyceraldehyde has aldehyde (C=O at top). DHAP has ketone (C=O in middle) + phosphate.

Where each molecule appears in different metabolic pathways

💧 Glycerol in lipid metabolism

Triglycerides → Glycerol → Glycerol-3-P → DHAP → Glycolysis

Glycerol kinase phosphorylates glycerol (liver, kidney — NOT in adipose tissue)

📊 Glyceraldehyde-3-P (G3P) in glycolysis

Glucose → Fructose-1,6-BP → G3P → 1,3-BPG → Pyruvate

G3P is the first intermediate that makes ATP in glycolysis. Also appears in Calvin cycle (photosynthesis).

🔀 DHAP at the crossroads

DHAP ⟷ G3P

Triose phosphate isomerase (TPI) — one of the fastest enzymes

DHAP can go to:

• Glycerol-3-P (lipid synthesis)
• G3P (continue glycolysis)

G3P only goes to:

• Forward in glycolysis
• Back to DHAP (reversible)

🏭 Triglyceride synthesis

DHAP → Glycerol-3-P + Fatty acyl-CoA → Triglyceride

DHAP is the precursor for glycerol backbone in all tissues. Glycerol itself can only be used in liver/kidney.

Key insight: DHAP is the metabolic hub. It connects glycolysis (carbs) to triglyceride synthesis (lipids). Glycerol comes from fat breakdown and feeds back into glycolysis.

Common confusion to avoid:

Glycerol-3-P ≠ G3P (glyceraldehyde-3-P)
Different molecules! Glycerol-3-P has no carbonyl (C=O), it's from lipid metabolism. G3P has an aldehyde, it's from glycolysis. They interconvert through DHAP as an intermediate.

Decision tree for metabolism:

If you see "triglyceride breakdown":

→ Produces glycerol + 3 fatty acids
→ Glycerol goes to liver/kidney → glycerol-3-P → DHAP → glycolysis

If you see "glucose → make fat":

→ Glycolysis produces DHAP
→ DHAP → glycerol-3-P (provides backbone)
→ Acetyl-CoA → fatty acids → attach to backbone

If you see "glycerol shuttle":

→ That's DHAP ⟷ glycerol-3-P interconversion
→ Used to move NADH equivalents into mitochondria
