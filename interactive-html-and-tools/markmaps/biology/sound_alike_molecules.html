Sound-Alike Molecules: Untangling the Confusion

MCAT loves testing these! Learn the patterns, not just the names.

⚠️ Why These Sound Similar (And Why MCAT Tests This)
The naming follows chemistry rules: -ine = base alone, -osine/-idine = base + sugar (nucleoside), -ate = nucleoside + phosphate(s).
But amino acids ALSO end in -ine (cysteine, glycine), and organs can sound similar (thymus).
Your job: Know which category each belongs to!

🧠 Thymine Family Memory
Thymine = just the base (like "Adam" = just the person)

🎯 Master Mnemonic: The Naming Pattern
For Nucleotides:
Base alone = -ine (Adenine, Guanine) Base + Sugar = -osine or -idine (Adenosine, Cytidine) + Phosphate(s) = -MP/-DP/-TP (AMP, GTP)

For Amino Acids:
If it builds proteins and ends in -ine, it's an amino acid (Cysteine, Glycine)

Context is KING:
DNA question + "-ine" word → Probably a base
Protein question + "-ine" word → Probably an amino acid
Immune system + "Thy-" → Probably the thymus organ
