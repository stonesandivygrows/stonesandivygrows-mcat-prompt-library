CARS shorthand

MI = main idea / walkaway
AUTH = author view
CON = contrast
EV = evidence line
EXT = new scenario
TRAP = unsupported

Behavioral shorthand

DEF = term/theory
IV/DV = variables
CORR != CAUSE
GRAPH = table/figure
DIR = direction
DEV = developmental stage
