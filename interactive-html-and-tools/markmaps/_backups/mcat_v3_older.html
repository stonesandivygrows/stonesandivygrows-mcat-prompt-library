UWorld protocol

  • Always start with 10–15 incorrects/omitted as warmup
  • Read every explanation — even correct answers
  • Error log: 1 line per wrong answer (principle or formula)
  • UWorld CARS = skip entirely (463 questions, not your CARS source)
  • Section priority: OrgoChem every Mon+Fri · Physics Wed · Biochem Tue · Bio Sat

CARS protocol

  • AAMC CARS Vol 1 = primary (2 passages/night on study days)
  • Jack Westin = 1 passage/night as supplement
  • After each session: categorize wrong answers by type (5 min) — this is the improvement log
  • 4.5 min passage read is fine — the clock issue is UWorld's timer methodology, not yours
  • Skip CARS on FL exam days, 1 max on pharmacy days

When to use each format — podcast vs MacWhisper summary vs transcript

🎧 Full Podcast — 8–11 AM (passive)

Play during the scroll/snooze window. Half-awake is fine — this is encoding, not active study. All podcasts are already made. Press play on today's section subjects. This primer makes the UWorld questions feel familiar at noon. Do not read or annotate during this — that defeats the passive benefit.

⚡ MacWhisper Summary — pre-UWorld (15 min)

Read the summary for today's subjects right before opening UWorld. This activates the content in working memory so you're not starting the questions cold. Also use post-dinner as a light second exposure before CARS or Anki. Summary = primer, not replacement. Best use of the summary feature.

📄 Transcript — on-demand reference

Only open when you get a UWorld question wrong and don't understand the explanation. Search the relevant concept in the transcript for that chapter. Targeted reference, not passive reading. Also useful when making an error log entry — pull the exact passage to understand the mechanism. Not a daily task.

The invisible labor — explicitly named and timed
#1 Error Log
During UWorld explanation review. 1 line per wrong answer: the principle or formula that was tested. Happens in real time — not a separate session. Takes ~2 min/wrong answer. This is how content sticks.
#2 CARS Improvement Log
5 min after each CARS session. Categorize wrong answers by type: inference / author's intent / main idea / application / EXCEPT. After 2 weeks: pattern emerges → targeted practice on that type. This is the CARS improvement guide.
#3 Day Review + Plan
15 min before CARS each night. Write: what was hardest, which subject needs more UWorld tomorrow, what formula appeared in multiple wrong answers, which podcast chapter is next. This means tomorrow's noon block starts in 30 seconds, not 45 minutes.
#4 MacWhisper Pre-Block
15 min pre-UWorld. Read today's podcast summary for each subject you're doing. This is not review — it's priming. The summary loads the vocabulary and concepts into working memory so the first 10 UWorld questions don't feel cold.
