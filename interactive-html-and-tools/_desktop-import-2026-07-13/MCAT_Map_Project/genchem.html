How to use this map

  • Pan: Drag on the map area with your mouse or finger (trackpad: click and drag). On a keyboard, use the arrow keys.
  • Zoom: Use the + and − buttons at the bottom-left, pinch on a trackpad, or keyboard + and − (zoom in/out).
  • Click a node: Highlights it and its connections; details appear in the right-hand panel.
  • Click empty space on the map: Clears highlights and resets the panel to the starting message.
  • Layout: Topics are arranged as 12 spokes from the center (clockwise from top). Most edges run along a spoke; faint dashed links are light cross-connections.
  • Fit: Zooms so the whole map is visible.
  • Clear: Removes selection highlights and resets the panel (same as clicking empty space).

Search: Use the search box in the header to find nodes by label or text in their descriptions. Matches are highlighted on the map; others fade. Click a result to zoom there and open the side panel. Use Clear search or Esc while the search box is focused to clear only the search.

Map key: Use the Key button for category colors and edge meanings.

Keyboard shortcuts (when you are not typing in a field): arrow keys pan; + / − zoom; 0 fits the map; Esc clears selection (and closes this help or the key panel).
