# General Chemistry Audio Overview Follow-Along Poster Plan

Purpose: Store the chapter-by-chapter production plan for MCAT General Chemistry audio-overview follow-along posters.

This file is meant to guide future poster generation from timestamped NotebookLM/audio overview transcripts. It does not replace the transcripts. Each chapter should still be planned from the transcript order once the timestamped transcript is available.

## Global Visual System

Use the same visual language as the generated Organic Chemistry Chapters 6-9 poster sets, adapted for General Chemistry.

### Standard poster set per chapter

For each chapter, generate:

1. Overall chapter poster/map
2. Subsection posters matching Kaplan chapter spine or lecture spine
3. Timestamped follow-along visuals in lecture order
4. Rescue visuals for dense concepts
5. Analogy/mnemonic cards when the audio uses memorable teaching hooks
6. MCAT trap/confusion cards
7. Optional slide deck only if the chapter is too equation-heavy for loose posters

### General Chemistry poster style

- Portrait infographic format
- White/light background
- Bold navy title/header system
- Color-coded rounded panels
- Dense but readable MCAT study-sheet layout
- Equations shown with variable definitions and units
- Proportional reasoning shown visually with arrows, not just formulas
- Graphs shown when relevant, especially slopes, axes, and curve meaning
- MCAT math shortcuts shown near equations
- Experiment setup diagrams shown when relevant
- Bottom MCAT Trap Zone strip
- Bottom High-Yield Takeaways strip
- Subsection labels visible on every generated poster

### General Chemistry visual rules

- Always show formulas with units and variable meanings
- Show what increases, decreases, or stays constant when one variable changes
- Include dimensional analysis when the audio discusses calculations
- Include graph-reading cues: axes, slope, intercept, asymptote, area under curve
- Include particle-level diagrams for atomic, bonding, gases, solutions, equilibrium, acids/bases, and electrochemistry
- Include reagent/condition arrows only when the topic involves chemical reactions
- Include real MCAT traps as bottom strips, not hidden in paragraphs

### Transcript-first rule

When a timestamped transcript is available:

- Follow the transcript order, not a rearranged textbook order
- Preserve exact timestamps
- Split any timestamp that covers more than one visual idea
- Preserve lecture analogies/mnemonics as cards
- Infer intended General Chemistry terms when the transcript has spelling errors
- Include every visualizable equation, graph, apparatus, particle model, comparison, or MCAT trap mentioned

### File naming convention

Use this naming pattern for generated poster files:

- Radiation panel: wavelength, frequency, energy; c = lambda f; E = hf

- A/Z isotope notation walkthrough
- Photon equation triangle: c = lambda f and E = hf

## MCAT trap cards

- Atomic number = protons, not protons plus neutrons
- Mass number = protons plus neutrons
- Isotopes differ by neutrons, ions differ by electrons
- Higher frequency means higher energy and shorter wavelength
- Electrons are removed from the highest principal energy level first in many ionization cases
