# Organic Chemistry Audio Overview Follow-Along Poster Plan

Purpose: Store the chapter-by-chapter production plan for MCAT Organic Chemistry audio-overview follow-along posters.

This file is meant to guide future poster generation from timestamped NotebookLM/audio overview transcripts. It does not replace the transcripts. Each chapter should still be planned from the transcript order once the timestamped transcript is available.

## Global Visual System

Use the same visual language as the generated Organic Chemistry Chapters 6-9 poster sets.

### Standard poster set per chapter

For each chapter, generate:

1. Overall chapter poster/map
2. Subsection posters matching Kaplan chapter spine
3. Timestamped follow-along visuals in lecture order
4. Rescue visuals for dense concepts
5. Analogy/mnemonic cards when the audio uses memorable teaching hooks
6. MCAT trap/confusion cards
7. Optional slide deck only if the chapter is too dense for loose posters

### Global poster style

- Portrait infographic format
- White/light background
- Bold navy title/header system
- Color-coded rounded panels
- Dense but readable MCAT study-sheet layout
- Structures and functional groups shown whenever relevant
- Substrate -> reagent/condition -> product reaction panels
- Curved-arrow mechanism panels when relevant
- Small timestamp badges on every section or visual block
- Bottom MCAT Trap Zone strip
- Bottom High-Yield Takeaways strip
- Consistent subsection labels, e.g. 6.1, 6.2, 6.3

### Transcript-first rule

When a timestamped transcript is available:

- Follow the transcript order, not a rearranged textbook order
- Preserve exact timestamps
- Split any timestamp that covers more than one visual idea
- Preserve lecture analogies/mnemonics as cards
- Infer intended organic chemistry terms when the transcript has spelling errors
- Include every visualizable structure, reaction, mechanism, reagent, trap, or comparison mentioned

### File naming convention

Use this naming pattern for generated poster files:

```text
orgo_ch##_overall_[short_title].png
orgo_ch##_section_##_[section_title].png
orgo_ch##_rescue_[concept].png
orgo_ch##_trap_[concept].png
```

Examples:

```text
orgo_ch06_overall_aldehydes_ketones_i.png
orgo_ch07_section_7_2_enolate_chemistry.png
orgo_ch08_rescue_decarboxylation.png
orgo_ch09_trap_derivative_reactivity_hierarchy.png
```

# Chapter 1: Nomenclature

## Chapter goal

Help the audio listener name organic molecules quickly without stopping to reconstruct parent chains, substituent priorities, numbering logic, stereochemistry labels, or common naming exceptions.

## Recommended poster set

6. Trap card: Common MCAT naming mistakes

## Overall poster layout

- Top roadmap: Identify functional group -> choose parent chain -> number chain -> name substituents -> assign stereochemistry -> assemble final name
- Central decision tree: suffix vs prefix priority
- Side table: functional group priority ladder
- Structure examples: alkanes, alkenes, alkynes, alcohols, carbonyls, carboxylic acids, amines, aromatics
- Naming construction strip: locants + substituents + parent + suffix
- Bottom trap zone: wrong parent chain, wrong numbering direction, forgetting stereochemistry, confusing common names

## Chronological follow-along visual targets

- Organic naming as a compression system
- Parent chain selection
- Functional group priority
- Numbering rules
- Substituent naming
- Alkyl/halo naming
- Alkene/alkyne locants
- Cycloalkanes and aromatic rings
- Cis/trans and E/Z labels
- R/S labels if included in transcript
- Common names and MCAT shorthand

## Rescue visuals

- Functional group priority ladder
- Parent-chain selection walkthrough
- Full name assembly template
- E/Z vs cis/trans
- R/S assignment workflow

## MCAT trap cards

- Longest chain is not always the correct parent if a higher-priority functional group is outside it
- Lowest-number rule prioritizes functional group over substituent
- Alphabetizing ignores di-, tri-, tetra-
- Common names can appear in passages even if IUPAC is not asked
- Cis/trans is not automatically E/Z

# Chapter 2: Isomers

## Chapter goal

Make isomer relationships visually obvious so the listener can classify two structures as constitutional isomers, conformers, enantiomers, diastereomers, meso compounds, or identical molecules.

## Recommended poster set

1. Overall poster: Organic Isomer Classification Map
2. Section poster: Constitutional vs stereoisomers
3. Section poster: Conformations and Newman projections
4. Section poster: Chirality, enantiomers, diastereomers, meso
5. Rescue card: R/S and mirror-image decision tree
6. Trap card: Identical vs enantiomer vs diastereomer

## Overall poster layout

- Top decision tree: Same formula? Same connectivity? Same 3D arrangement?
- Left branch: constitutional isomers
- Middle branch: conformers
- Right branch: configurational stereoisomers
- Central comparison grid: identical, constitutional, conformational, enantiomeric, diastereomeric, meso
- Bottom optical activity strip

## Chronological follow-along visual targets

- Structural formula vs molecular formula
- Constitutional isomers
- Conformational isomers
- Newman projections
- Staggered/eclipsed conformations
- Anti/gauche relationships
- Chiral centers
- Enantiomers and mirror images
- Diastereomers
- Meso compounds
- Optical activity and racemic mixtures

## Rescue visuals

- Isomer decision tree
- Newman projection rotation/rescue visual
- Chirality center recognition map
- Meso compound internal plane of symmetry
- R/S assignment workflow

## MCAT trap cards

- Rotations around single bonds do not create new configurational isomers
- Mirror image does not automatically mean enantiomer if the molecule is achiral or meso
- Same R/S at every center = same molecule
- Opposite R/S at every center = enantiomer only if all stereocenters invert
- Mixed R/S changes = diastereomer

# Chapter 5: Alcohols

## Chapter goal

Show alcohols as both functional groups and reaction hubs: acidic/basic behavior, oxidation level, substitution, protection, and biological relevance.

## Recommended poster set

1. Overall poster: Alcohols as Organic Reaction Hubs
2. Section poster: Alcohol properties and acidity
3. Section poster: Oxidation of alcohols
4. Section poster: Substitution/dehydration/protection reactions
5. Rescue card: Primary vs secondary vs tertiary alcohol reactions
6. Trap card: PCC vs strong oxidizers

## Overall poster layout

- Top: R-OH structure and classification
- Properties: hydrogen bonding, boiling point, solubility
- Oxidation ladder: 1° alcohol -> aldehyde -> carboxylic acid; 2° alcohol -> ketone; 3° alcohol no ordinary oxidation
- Reagent panel: PCC, chromic acid, KMnO4, SOCl2/PBr3, acid dehydration
- Protection panel: silyl ethers if included
- Bottom trap zone: oxidizer strength, leaving group conversion, dehydration vs substitution

## Chronological follow-along visual targets

- Alcohol structure and classification
- Hydrogen bonding and physical properties
- Acidity of alcohols
- Conversion to alkoxides
- Oxidation reactions
- Reduction connections to carbonyl chemistry
- Alcohol as poor leaving group unless protonated/converted
- Substitution and elimination reactions
- Biological alcohol examples if mentioned

## Rescue visuals

- Alcohol classification tree
- Oxidation ladder with reagents
- Leaving group activation workflow
- Dehydration mechanism
- Alcohol vs phenol acidity comparison

## MCAT trap cards

- Tertiary alcohols resist oxidation without C-C bond cleavage
- PCC stops primary alcohols at aldehydes
- Strong oxidizers continue primary alcohol oxidation to carboxylic acids
- OH is a poor leaving group until protonated or converted
- Phenols are more acidic than ordinary alcohols because phenoxide is resonance-stabilized

- Reaction logic: nucleophilic nitrogen, protonation state, phosphoryl transfer
- Bottom trap strip: amine vs amide, phosphate charge, ATP hydrolysis, resonance/basicity

## Chronological follow-along visual targets

- Amine classification: primary, secondary, tertiary, quaternary ammonium
- Amine basicity and protonation

- Imine/enamine if included

- Nitrogen functional group atlas
- Amine vs amide basicity comparison
- Amino acid zwitterion charge map
- Phosphate pKa/charge state map
- Nucleotide scaffold visual: base + sugar + phosphate
- ATP as nucleotide triphosphate energy carrier

## MCAT trap cards

- Amides are much less basic than amines because the nitrogen lone pair is delocalized into the carbonyl
- Quaternary ammonium is permanently charged
- Amino acids can be zwitterions at physiological pH
- Phosphate groups are usually negatively charged near physiological pH
- ATP is not just energy; it is a nucleotide triphosphate with adenine, ribose, and phosphates
- Phosphodiester bonds form the nucleic acid backbone

# Chapter 11: Spectroscopy

## Chapter goal

Create a structure-solving visual workflow for IR, NMR, UV-vis, and mass spectrometry so the listener can interpret spectra without memorizing isolated numbers.

## Recommended poster set

1. Overall poster: Spectroscopy Structure-Solving Workflow
2. Section poster: IR functional group identification

5. Section poster: UV-vis and conjugation
6. Section poster: Mass spectrometry and molecular fragments
7. Rescue card: How to solve a spectrum step by step

## Overall poster layout

- Top workflow: formula -> DBE -> IR -> NMR -> MS -> final structure
- IR panel: broad OH/NH, sharp carbonyl, C-H stretches, nitrile, alkene/alkyne
- NMR panel: chemical shift map, integration, n+1 splitting, equivalent hydrogens
- MS panel: molecular ion, base peak, common fragments
- UV-vis panel: conjugation increases wavelength
- Bottom trap strip: broad OH vs NH, aldehyde H, equivalent protons, splitting neighbors

## Chronological follow-along visual targets

- Purpose of spectroscopy
- Electromagnetic radiation and absorption
- IR: bond vibration and functional group peaks
- NMR: shielding/deshielding, chemical shifts
- Integration
- Splitting and neighboring hydrogens
- Equivalent hydrogens
- 13C NMR basics
- UV-vis and conjugation
- Mass spectrometry molecular ion/base peak/fragments
- Combined spectrum workflow

## Rescue visuals

- IR diagnostic peak map
- NMR chemical shift ruler
- Splitting tree with n+1 rule
- Equivalent proton recognition
- DBE/degree of unsaturation workflow
- Combined unknown-structure solving flowchart

## MCAT trap cards

- Broad OH peaks can obscure other signals
- Aldehyde protons appear far downfield
- Integration tells relative number of hydrogens, not absolute count unless scaled
- Equivalent hydrogens do not split each other
- More conjugation shifts UV-vis absorption to longer wavelength
- Base peak is most abundant fragment, not necessarily molecular ion

# Chapter 12: Separations and Purifications

## Chapter goal

Make separation methods visually operational: what property is being separated, what moves fastest, what stays behind, and what MCAT trap distinguishes each method.

## Recommended poster set

1. Overall poster: Organic Separation and Purification Decision Tree
2. Section poster: Extraction and acid-base separation
3. Section poster: Chromatography family
4. Section poster: Distillation and boiling point separation
5. Section poster: Electrophoresis and charge/size separation
6. Section poster: Recrystallization and purification logic
7. Rescue card: Which technique should I choose?
8. Trap card: Stationary vs mobile phase confusion

## Overall poster layout

- Top decision tree: separate by solubility, boiling point, polarity, size, charge, affinity
- Extraction panel: aqueous vs organic layers, acid/base protonation state
- Chromatography panel: TLC, column, HPLC, gas chromatography
- Distillation panel: simple vs fractional vs vacuum
- Electrophoresis panel: gel movement by charge/size
- Recrystallization panel: dissolve hot, crystallize cold, impurities remain in solution
- Bottom trap zone: Rf, polarity, layer identity, ionized vs neutral form, boiling point vs vapor pressure

## Chronological follow-along visual targets

- Purpose of separations
- Extraction setup
- Acid-base extraction
- Solubility and partitioning
- Chromatography principles
- TLC and Rf
- Column chromatography
- HPLC/GC if included
- Distillation types
- Recrystallization
- Electrophoresis
- Purity and yield logic

## Rescue visuals

- Separation method decision tree
- Acid-base extraction flowchart
- TLC plate and Rf calculation
- Chromatography polarity comparison
- Distillation apparatus comparison
- Gel electrophoresis migration map

## MCAT trap cards

- Neutral molecules usually partition into organic layer; charged molecules usually prefer aqueous layer
- The organic layer is not always the top layer
- Higher Rf means stronger interaction with mobile phase or weaker interaction with stationary phase
- In normal-phase chromatography, more polar compounds stick more to silica and move slower
- Fractional distillation is better for close boiling points
- Smaller DNA fragments travel farther in gel electrophoresis
