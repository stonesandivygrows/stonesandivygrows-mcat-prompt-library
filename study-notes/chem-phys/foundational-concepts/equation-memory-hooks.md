# Equation Memory Hooks

## Planck's Constant

\[
h \approx 6.6 \times 10^{-34}\; J\cdot s
\]

MCAT rounding:

\[
h \approx 7 \times 10^{-34}
\]

Memory hook:

- Planck = photon-sized quantity
- 7 × 10^-34
- Do not confuse with 10^-32
- Multiplying by c (3 × 10^8) gives hc ≈ 2 × 10^-25, which helps anchor the exponent

## Centripetal Acceleration

\[
a_c=\frac{v^2}{r}
\]

Memory hook:

- Speed matters twice:
  - more velocity to redirect
  - less time to redirect it
- Bigger radius softens the turn

Translation:

- Faster object = much larger centripetal acceleration
- Larger circle = smaller centripetal acceleration

## Parts Per Million (ppm)

\[
ppm=\frac{solute}{solution}\times 10^6
\]

Memory hook:

- Same units cancel
- 10^6 means 'per million'
- Not a mega-prefix problem

Important:

- Formal definition uses solution in denominator
- For dilute solutions:

\[
solution \approx solvent
\]

which is why mg/L ≈ ppm in water.
