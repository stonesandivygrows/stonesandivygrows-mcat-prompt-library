# Global Rules

## Organization Rules
- Organize primarily by Kaplan subject → chapter → section.
- Use transcript timestamps as metadata only.
- Do not organize the main file by transcript order.
- Merge duplicates instead of repeating concepts.
- Use major concept entries for important ideas.
- Use micro-nodes for smaller details.

## Concept Web Rules
- This is a web, not a flowchart.
- Every important relationship should be labeled.
- Do not use vague relationship labels like “related to” or “connected to.”
- Prefer precise labels like causes, requires, increases, decreases, regulates, converts to, opposes, maintains, explains, is tested by, is confused with.

## Transcript Rules
- Correct obvious speech-to-text errors silently using MCAT context.
- Do not invent facts.
- If something is unclear, mark it as VERIFY.
- If the transcript appears to combine multiple chapters, ask before making major changes.
