# Final MCAT Stretch Feasible Plan

**Window:** 2026-06-10 to 2026-06-27  
**Target test date:** 2026-06-27  
**Known reduced-availability days:** 2026-06-18 and 2026-06-24  
**Current-cycle full-lengths completed:** AAMC Practice Exam 1 and UWorld Exam 1  
**Planning principle:** score repair over resource completion.

---

## Core rules

1. **Progress is counted by completed assignments, not hours studied.**
2. **Questions + review are the spine.**
3. **CARS is a daily gym workout, not a bank to finish.**
4. **Equations/units are a daily rotation, not passive rereading.**
5. **MCATalyst is targeted content repair plus recall output.**
6. **MCATalyst Anki content can be memorized through any medium, not only Anki.**
7. **Mind maps are struggle-triggered only.**
8. **Whiteboard practice must happen during actual questions.**
9. **Past UWorld tests are used to prove repair, not to reread old PDFs.**
10. **No catch-up debt after a red/reduced day.**

## Priority ranking

### P0 — Full-length exam / full-length review
**Use when scheduled.**

Completion:
- Full-length day = take exam only.
- Review day = review 1–2 sections, redo 15 missed/shaky questions, write max 5 repair rules.

Do not add normal UWorld volume on full-length days.

### P1 — Current AAMC + UWorld question review
**Normal-day assignment:** 30–44 UWorld science questions total.

Default split:
- Bio: 8 Q
- Biochem: 8 Q
- Gen Chem: 8 Q
- Physics: 8 Q
- Orgo: 4 Q or 1 reaction/lab drill
- P/S: 8 Q or 25 terms

Review completion:
- Review all wrong questions.
- Review all flagged questions.
- Review guessed-correct questions.
- Retake missed/shaky questions from yesterday.
- Write max 5 repair rules.

Minimum version:
- 20 science questions total from weakest subjects.
- Review only wrong + flagged.
- Retake 5 missed/shaky questions.

---

### P1.5 — Past UWorld test repair
**Purpose:** prove old mistakes are repaired.

Assignment 2–3x/week:
- Retake 15–25 missed/shaky questions from an old UWorld test.
- Submit if possible.
- Review only questions still missed, guessed, or answered by memory.
- Write max 3 repair rules.

Priority order:
1. UWorld Exam 1 review leftovers
2. UWorld Chemistry/Biochem 59Q tests
3. UWorld Test 2 retake pending
4. UWorld Test 12 repaired concepts
5. UWorld Test 23 P/S terms
6. Any test where the same weakness keeps appearing

Minimum version:
- Retake 5 old missed questions.
- Write 1 repair rule.

### P2 — CARS gym workout
**Normal assignment:** 1–2 passages.

Resources:
- UWorld CARS
- AAMC CARS QPacks / Diagnostic Tool

Completion:
- Do passage(s).
- Review missed questions.
- Review 50/50 questions.
- Label the trap type.

Minimum version:
- 1 passage.
- Review missed only.

Bonus version:
- 3 passages if sharp.
- Stop before turning CARS into a full autopsy rabbit hole.

### P3 — Equations and units
**Normal assignment:** 2 pages from **MCAT_Equations_Worksheets (1).pdf**.

Rotation:
- Day A: Worksheet 1 — Equations Drill
- Day B: Worksheet 2 — Units Drill
- Day C: Worksheet 3 — Variable Connections
- Day D: Worksheet 1 — Equations Drill
- Day E: Worksheet 4 — Symbol Disambiguation
- Day F: Redo missed pages only
- Day G: Constants, prefixes, and units recall

Completion:
- Fill page from memory first.
- Check with MCATalyst equations only after recall.
- Redo missed formulas/units once.

Minimum version:
- 1 page or 10 formulas/units from memory.

---

### P3.5 — Whiteboard practice
**Purpose:** practice test-day writing habits.

Normal assignment:
- 5-minute setup before C/P or B/B question block.
- Use during actual questions.
- Audit whether it helped.

Choose one daily:
- C/P equation/constants setup
- Unit conversion setup
- Stoichiometry GIVEN → WANT path
- Redox OIL RIG / oxidizing vs reducing agent
- CARS timing checkpoints
- B/B passage map with 2–3 variables only
- 6–7–8 arithmetic box
- Physics signs/units box

Weekly assignment:
- 2x/week: do one timed UWorld/AAMC block using the whiteboard exactly like test day.

Minimum version:
- Write constants/prefixes/equation skeleton once.
- Do 5 questions using that whiteboard.

---

### P4 — Daily question / repair block
**Normal assignment:** resolve 1 real question fully.

Resources:
- ChatGPT
- UWorld explanation
- AAMC explanation
- MCATalyst content doc
- Whiteboard / iPad / paper

Completion output — choose one:
- 5-sentence explanation
- wrong thought → corrected thought
- mini flowchart
- tiny mind map
- 3 Anki cards
- 1 sticky-note rule
- 1 worked example

Rule:
If this takes about an hour, it replaces extra MCATalyst reading or extra Anki. It does not get added on top.

---

### P5 — Flexible memory set / MCATalyst Anki content
**Normal assignment:** complete 1 memory set.

Choose one medium:
- Anki app: 60–100 cards, capped
- Printed Anki PDF: 2–4 pages active recall
- Paper copying: 1 page copied from memory, then corrected
- iPad drawing: 1 diagram/pathway/reaction set redrawn
- Sticky notes: 10–15 high-yield facts
- Physical flashcards: 15–25 cards made or reviewed
- Poster paper: 1 mini poster for a weak topic
- Reaction drawing: 1 reaction family or mechanism ladder from memory
- Infographic review: redraw or annotate 1 MCATalyst graphic
- Mind map: only if struggle-triggered

Minimum version:
- 20 cards or 1 small diagram or 5 sticky notes.

---

### P6 — MCATalyst content doc
**Normal assignment:** 4–6 pages or 1 small subsection.

Completion:
- Read the assigned pages/subsection.
- Close the document.
- Write 5 recall bullets.
- Write 3 facts/equations/relationships to memorize.
- Connect to 1 missed UWorld/AAMC question or struggle topic.

Minimum version:
- 2 pages from a problem area.
- 3 recall bullets.

Heavy repair version:
- 8–12 pages max from one problem area.
- End with mini quiz, diagram, or flashcards.

---

### P7 — Mind maps
**Normal assignment:** not required.

Trigger only if:
- Same concept missed twice.
- Topic takes a long time to understand.
- Variables/equations blend together.
- Pathways/reactions are not sticking.
- Same question keeps recurring.

Do not mind map:
- Topics already known.
- Simple facts.
- One-off errors.
- Anything fixed by one sentence.

## Feasibility audit built into the plan

### If behind
- Do not catch up by doubling tomorrow.
- Return to the next scheduled day.
- Use missed work as optional bonus only.

### If finished early
Choose 1–2 only:
- Redo 10 missed UWorld questions.
- Redo 5 AAMC missed/flagged questions.
- Do 1 extra CARS passage.
- Redo 1 missed equation worksheet page.
- Draw one pathway/reaction/infographic.
- Make 5 repair rules from old mistakes.

### If procrastinating
Choose one tiny action:
- 5 UWorld questions.
- 1 CARS passage untimed.
- 1 equation page.
- 20 cards or 5 sticky notes.
- 2 MCATalyst pages + 3 recall bullets.
- Ask one daily question.

### If a concept takes an hour
That is the repair block for the day.
Cut extra MCATalyst, extra Anki, and bonus questions first.
Do not cut review of today’s wrong questions.
